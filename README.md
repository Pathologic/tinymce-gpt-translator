# tinymce-gpt-translator
Tinymce plugin to translate text with GPT

Add `gpttranslator` to the plugins list, add the same to the buttons list.
Add GPT parameters to the plugin custom parameters:
```
 gpt_api_key: 'your API key',
 gpt_model: 'gpt-5-mini',
 gpt_api_url: 'https://bothub.chat/api/v2/openai/v1/chat/completions',
```
